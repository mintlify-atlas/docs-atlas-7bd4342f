Git can be installed on the most common operating systems like Windows, Mac, and Linux. In fact, Git comes installed by default on most Mac and Linux machines!

## Checking for Git

To see if you already have Git installed, open up your terminal application.

<Steps>
  <Step title="Open your terminal">
    - **Mac**: Look for a command prompt application called "Terminal"
    - **Windows**: Open the windows command prompt or "Git Bash"
  </Step>
  
  <Step title="Check the Git version">
    Type the following command:
    
    ```bash
    git version
    ```
    
    The output will either tell you which version of Git is installed, or it will alert you that `git` is an unknown command.
  </Step>
</Steps>

<Note>
If `git` is an unknown command, read further to find out how to install Git for your operating system.
</Note>

## Install Git on Windows

<Steps>
  <Step title="Download the installer">
    Navigate to the latest [Git for Windows installer](https://gitforwindows.org/) and download the latest version.
  </Step>
  
  <Step title="Run the installer">
    Once the installer has started, follow the instructions as provided in the **Git Setup** wizard screen until the installation is complete.
  </Step>
  
  <Step title="Open command prompt">
    Open the windows command prompt (or **Git Bash** if you selected not to use the standard Git Windows Command Prompt during the Git installation).
  </Step>
  
  <Step title="Verify installation">
    Type `git version` to verify Git was installed.
    
    ```bash
    git version
    ```
  </Step>
</Steps>

<Tip>
[`git-scm`](https://git-scm.com/download/win) is a popular and recommended resource for downloading Git for Windows. The advantage of downloading Git from `git-scm` is that your download automatically starts with the latest version of Git included with the recommended command prompt, `Git Bash`.
</Tip>

### Install Git on Windows through Visual Studio Code

GitHub integration is provided through the [GitHub Pull Requests and Issues extension](https://marketplace.visualstudio.com/items?itemName=GitHub.vscode-pull-request-github).

<Steps>
  <Step title="Install the extension">
    Install the GitHub Pull Requests and Issues extension in VS Code.
  </Step>
  
  <Step title="Sign in to GitHub">
    Follow the prompts to authenticate with GitHub and return to VS Code.
  </Step>
  
  <Step title="Start using Git in VS Code">
    You can perform actions like searching for and cloning a repository from GitHub using the **Git: Clone** command in the Command Palette (Ctrl+Shift+P) or by using the Clone Repository button in the Source Control view.
  </Step>
</Steps>

## Install Git on Mac

Most versions of MacOS will already have `Git` installed, and you can activate it through the terminal with `git version`. However, if you don't have Git installed, you can install the latest version using one of several popular methods:

### Install Git From an Installer

<Steps>
  <Step title="Download the installer">
    Navigate to the latest [macOS Git Installer](https://sourceforge.net/projects/git-osx-installer/files/git-2.23.0-intel-universal-mavericks.dmg/download?use_mirror=autoselect) and download the latest version.
  </Step>
  
  <Step title="Run the installer">
    Once the installer has started, follow the instructions as provided until the installation is complete.
  </Step>
  
  <Step title="Verify installation">
    Open the command prompt "terminal" and type:
    
    ```bash
    git version
    ```
  </Step>
</Steps>

<Tip>
[`git-scm`](https://git-scm.com/download/mac) is a popular and recommended resource for downloading Git on a Mac. The advantage of downloading Git from `git-scm` is that your download automatically starts with the latest version of Git.
</Tip>

### Install Git from Homebrew

[Homebrew](https://brew.sh/) is a popular package manager for macOS. If you already have Homebrew installed, you can follow these steps:

<Steps>
  <Step title="Install Git with Homebrew">
    Open up a terminal window and install Git using the following command:
    
    ```bash
    brew install git
    ```
  </Step>
  
  <Step title="Verify installation">
    Once the command output has been completed, verify the installation:
    
    ```bash
    git version
    ```
  </Step>
</Steps>

## Install Git on Linux

<Note>
Fun fact: Git was originally developed to version the Linux operating system! So, it only makes sense that it is easy to configure to run on Linux.
</Note>

You can install `Git` on Linux through the package management tool that comes with your distribution.

### Debian/Ubuntu

<Steps>
  <Step title="Update package list">
    Navigate to your command prompt shell and run the following command to make sure everything is up-to-date:
    
    ```bash
    sudo apt-get update
    ```
  </Step>
  
  <Step title="Install Git">
    Run the following command:
    
    ```bash
    sudo apt-get install git-all
    ```
  </Step>
  
  <Step title="Verify installation">
    Once the command output has been completed, verify the installation:
    
    ```bash
    git version
    ```
  </Step>
</Steps>

### Fedora

<Steps>
  <Step title="Install Git">
    Navigate to your command prompt shell and run:
    
    ```bash
    sudo dnf install git-all
    ```
  </Step>
  
  <Step title="Verify installation">
    Once the command output has been completed, verify the installation:
    
    ```bash
    git version
    ```
  </Step>
</Steps>

<Tip>
You can download the proper Git versions and read more about how to install on specific Linux systems in [git-scm's documentation](https://git-scm.com/download/linux).
</Tip>
